# ChatGPT Warm Minimal

Extension Microsoft Edge unpacked pour ameliorer l'interface web de ChatGPT.

## Ce que fait cette premiere version

- ajoute un panneau flottant qui liste les reponses en cours de reflexion ou de generation
- personnalise la barre laterale gauche, son en-tete, ses items et le profil
- personnalise le header principal et le selecteur de modele
- personnalise la barre de saisie et ses controles
- elargit la zone de lecture des reponses
- reduit legerement l'espacement des lignes dans les reponses

## Installation dans Microsoft Edge

1. Ouvrir `edge://extensions`
2. Activer `Mode developpeur`
3. Cliquer sur `Charger l'extension non empaquetee`
4. Selectionner le dossier `edge-chatgpt-warmth`

## Fichiers

- `manifest.json`
- `content.js`
- `styles.css`
